# CHALOUMI -- Social Media Content Package
## APEX DIGITAL | Mai 2026

### Structure

```
social-content/
  instagram/
    posts/         30 posts prets a publier (caption + hashtags + CTA)
    stories/       12 semaines de story prompts
  tiktok/
    scripts/       Scripts video complets avec timecodes
  pinterest/
    pins/          Descriptions optimisees SEO pour epingles
  calendrier_editorial_12semaines.md   Planning complet
  hashtag_sets.md                       Sets de hashtags pre-configures
  influencer_outreach/                  Templates et liste cible
```

### Comment utiliser

1. **Calendrier** : Suivre `calendrier_editorial_12semaines.md` pour le planning hebdomadaire
2. **Posts Instagram** : Copier-coller caption + hashtags depuis les fichiers individuels
3. **TikTok** : Utiliser les scripts comme guide de tournage (timecodes precis)
4. **Pinterest** : Descriptions SEO-optimisees, pretes pour Tailwind/Later
5. **Hashtags** : Sets pre-configures dans `hashtag_sets.md`, copier le set adapte

### Outils recommandes
- **Planification** : Later ou Planoly (15 EUR/mois)
- **Design** : Canva Pro (12 EUR/mois)
- **Montage video** : CapCut (gratuit)
- **Analytics** : Metricool (gratuit)

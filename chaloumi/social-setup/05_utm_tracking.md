## CONVENTIONS UTM

| Parametre | Convention |
|-----------|-----------|
| `utm_source` | Plateforme exacte : `instagram`, `tiktok`, `pinterest`, `facebook`, `meta`, `brevo`, `referral`, `ugc`, `linktree` |
| `utm_medium` | Type de canal : `social`, `paid_social`, `email`, `parrainage` |
| `utm_campaign` | Format : `[objectif]_[detail]` — ex: `awareness_m1`, `welcome_j1`, `pin` |
| `utm_content` | Identifiant du contenu : nom du post, nom de l'ad, section de l'email |
| `utm_term` | Reserve pour les mots-cles paid search (non utilise pour social) |

### Regles
- Tout en minuscules
- Tirets bas `_` pour separer les mots
- Pas d'accents
- Mois = m1, m2... m6
- Jours (email) = j0, j1, j3, j7...
- Remplacer `[variables]` par les valeurs reelles avant utilisation

## RACCOURCISSEUR

Pour les liens dans les bios et posts (quand le lien complet est trop long) :
- Utiliser **Bitly** ou **Rebrandly** avec domaine personnalise `lnk.chaloumi.fr`
- Toujours garder les UTM dans le lien long avant raccourcissement
- Le raccourcisseur preserve les parametres UTM dans la redirection

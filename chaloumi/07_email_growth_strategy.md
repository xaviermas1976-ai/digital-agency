# 3. GROWTH LOOPS

## 3.1 PROGRAMME PARRAINAGE

### Mecanique

| Element | Detail |
|---------|--------|
| Offre parrain | -5 EUR sur la prochaine commande (sans minimum) |
| Offre filleul | -15% sur la premiere commande (code unique genere) |
| Limite | 10 parrainages max par client (50 EUR max de credits) |
| Tracking | Code unique par parrain (format : AMI-PRENOM-XXXX) |
| Declencheur | Email PA3 (J+7 post-livraison) + page compte client |

### Message type parrain
> "J'ai decouvert un fromage de chevre artisanal incroyable, fait a Chypre. Ca s'appelle Chaloumi et ca n'a rien a voir avec le halloumi du supermarche. Goute avec mon code {CODE} et tu auras -15% sur ta premiere commande."

### Implementation
- Brevo : segment "clients actifs" + automation "post-achat J+7"
- Page dediee /parrainage sur le site avec formulaire (entrer email ami)
- Dashboard parrain dans l'espace client (credits cumules, statut invitations)
- Cout estime : 5 EUR par parrainage converti (budget 200-400 EUR sur 6 mois pour 40-80 parrainages)

### Projection
- Taux de parrainage : 8-12% des clients actifs parrainent
- Conversion filleul : 25-35% des filleuls commandent
- K-factor estime : 0.15-0.25 (contribution incrementale, pas viral pur)

---

## 3.2 VIRAL LOOP : PARTAGE RECETTE

### Mecanique
1. Client consulte une recette sur le blog
2. Bouton "Partager cette recette a un ami gourmand" (email ou WhatsApp)
3. L'ami recoit la recette + code RECETTE10 (-10% premiere commande)
4. Si l'ami commande, le partageur recoit -5 EUR credit

### Implementation
- Plugin partage social sur chaque page recette (WhatsApp prioritaire en France)
- UTM tracking : utm_source=recipe_share&utm_medium=whatsapp&utm_campaign={recipe_slug}
- Brevo : automation declenchee par UTM a l'inscription
- Cout : quasi nul (reduction conditionnelle)

### Projection
- 5-10 partages recette/semaine a regime
- Conversion partage vers achat : 8-12%
- 2-5 nouveaux clients/mois par ce canal

---

## 3.3 UGC LOOP : PHOTO CLIENT

### Mecanique
1. Client poste photo de son plat Chaloumi sur Instagram avec #MonChaloumi + tag @chaloumi
2. L'equipe reposte les meilleures photos en story + feed (1-2/semaine)
3. Chaque repost : le client recoit par DM un code -10% prochaine commande
4. Les photos alimentent la newsletter (section communaute) + les ads

### Implementation
- Veille hashtag #MonChaloumi (outil gratuit : notifications Instagram)
- Templates Canva pour repost story (cadre branding Chaloumi)
- DM automatise via ManyChat (optionnel, 15 EUR/mois) ou DM manuel
- Droits d'utilisation : mention dans bio "vos photos peuvent etre repartagees"

### Projection
- 3-5 UGC/semaine apres 3 mois
- Taux engagement stories UGC : 2-3x superieur aux photos de marque
- Cout : -10% sur commande ~30 EUR = 3 EUR/photo. Budget : 50-100 EUR/mois

## 5.3 POINTS DE FRICTION IDENTIFIES + SOLUTIONS

### Friction 1 : Prix percu eleve (fromage artisanal + livraison Chypre-France)
- **Solution** : Ancrage de valeur -- comparer au restaurant ("en restaurant, une portion Chaloumi coute 12 EUR. Chez nous, 500g = 10+ portions pour 25 EUR")
- **Solution** : Livraison gratuite des 35 EUR (seuil psychologique)
- **Solution** : Abonnement avec -12 a -18% vs achat unique

### Friction 2 : Meconnaissance du produit (confusion halloumi/Chaloumi)

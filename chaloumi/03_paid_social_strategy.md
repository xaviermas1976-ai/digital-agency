```

+-- [CBO] PHASE 1 - AWARENESS (Mois 1-2)
|   +-- Ad Set : Foodies Interet Large
|   +-- Ad Set : BBQ & Grillades
|   +-- Ad Set : Consom'acteurs Bio/Local
|   +-- Ad Set : Lookalike 1% Email List (si dispo)
|
+-- [CBO] PHASE 2 - CONSIDERATION (Mois 2-4)
|   +-- Ad Set : Video Viewers 50%+ (Custom)
|   +-- Ad Set : Page/Insta Engagers 30j
|   +-- Ad Set : LP Visitors non-acheteurs
|   +-- Ad Set : Lookalike 1% Engagers
|
+-- [ABO] PHASE 3 - CONVERSION (Mois 3-6)
|   +-- Ad Set : Retargeting LP Visitors 7j
|   +-- Ad Set : Retargeting Add-to-Cart 14j
|   +-- Ad Set : Retargeting Video Viewers 75%+ 14j
|   +-- Ad Set : Lookalike 1% Acheteurs (des M3)
|
+-- [ABO] ALWAYS-ON - RETENTION (des M4)

```

**Pourquoi CBO en Phase 1-2, ABO en Phase 3 :** CBO laisse l'algo Meta trouver les poches de performance sur audiences froides. ABO en conversion donne un controle precis du budget sur les audiences chaudes a forte intention.

## 1.7 Regles d'optimisation

| Regle | Condition | Action |
|-------|-----------|--------|
| Kill ad | CTR < 0.8% apres 1000 impressions | Couper l'ad |
| Kill ad set | CPA > 20 EUR apres 50 EUR depenses | Couper l'ad set |
| Scale ad set | CPA < 10 EUR sur 3 jours consecutifs | +20% budget (pas plus de +20%/3j) |
| Refresh creative | Frequence > 2.5 sur audience froide | Nouveau creative, pause l'ancien |
| Shift budget | ROAS retargeting > 2x awareness | +15% vers retargeting |
| Audience fatigue | CPM +30% semaine/semaine sur meme audience | Elargir audience ou pause 7j |
| Advantage+ test | M3 si 50+ purchases | Lancer 1 campagne ASC avec creative gagnants |

# 3. RETARGETING -- Cross-plateforme

## 3.1 Setup technique requis

| Element | Priorite | Detail |
|---------|----------|--------|
| Meta Pixel | CRITIQUE | Installer sur LP avant tout lancement |
| Meta Conversions API (CAPI) | HAUTE | Server-side events via Shopify/WooCommerce plugin ou GTM server |
| TikTok Pixel | CRITIQUE | Installer avant M1 |
| TikTok Events API | MOYENNE | Implementer M2 pour ameliorer attribution |
| Evenements standards | CRITIQUE | PageView, ViewContent, AddToCart, InitiateCheckout, Purchase |
| Valeur d'evenement | HAUTE | Passer la valeur du panier dans Purchase event |
| UTM tagging | CRITIQUE | utm_source/medium/campaign/content sur chaque ad |

### Schema CAPI simplifie

```
Visiteur LP --> Pixel navigateur (client-side)
                    +
              CAPI serveur (server-side)
                    |
              Deduplication par event_id
                    |
              Meta / TikTok recoit 1 event fiable
```

# 4. CALENDRIER DE DEPLOIEMENT

## Mois 1-2 : Test & Learn

| Semaine | Action | Budget |
|---------|--------|-------:|
| S1 | Installation Pixel Meta + TikTok + CAPI. Lancement 3 ad sets Awareness Meta (3 creatives chacun = 9 ads). | 250 |
| S2 | Analyse premiers resultats. Kill ads CTR < 0.8%. Lancement TikTok Discovery (3 ad groups). | 250 |

| S4 | Bilan M1. Identifier top 2 audiences + top 3 creatives. | 225 |

### Matrice de test M1-M2

| Variable testee | Nombre de variants | Budget test | Critere de succes |
|-----------------|-------------------:|------------:|-------------------|
| Audiences (interets) | 4 | 400 | CTR > 1.5%, CPM < 10 EUR |
| Creative angles | 5 | 500 | Thumb-stop > 25%, CTR > 1.5% |
| Formats (Reels vs Carousel) | 2 | 200 | CPC comparatif |
| Copy (long vs court) | 2 | 100 | CTR comparatif |
| CTA (Acheter vs Decouvrir) | 2 | 100 | CTR comparatif |

**Regle de decision :** Minimum 1000 impressions et 5 EUR depenses avant de juger un ad. Minimum 50 EUR depenses avant de juger un ad set.

## Mois 3-4 : Scale

| Critere pour scaler | Seuil |
|---------------------|-------|
| CPA Meta < 15 EUR sur 7 jours | Scale +20% budget ad set |
| ROAS Meta > 1.5x sur 7 jours | Scale +20% |
| CTR > 2% + CPC < 0.50 EUR | Dupliquer ad set avec audience elargie |
| 50+ Purchase events | Lancer Advantage+ Shopping Campaign |
| TikTok CPA < 18 EUR | Augmenter budget TikTok +25% (prendre sur Meta awareness) |

**Actions M3-M4 :**
- Lancer Phase 3 Conversion Meta
- Activer retargeting sequences completes
- Creer Lookalike 1% acheteurs (si > 100 purchases)
- Tester 3-5 nouveaux creatives par plateforme
- Activer Advantage+ Shopping si eligible

## Mois 5-6 : Optimisation

| Objectif | Cible |
|----------|-------|
| CPA Meta | < 12 EUR |
| CPA TikTok | < 15 EUR |
| ROAS global | > 1.8x |
| Frequence retargeting 7j | < 4.0 |
| Creative refresh | Toutes les 2 semaines |

**Actions M5-M6 :**
- Budget shift : max budget vers audiences/creatives top 20% (loi 80/20)
- Retargeting = 20-25% du budget
- Retention campaign sur acheteurs (reorder, decouverte nouveau format)
- Bilan complet et recommandation M7+

# 7. RISQUES ET MITIGATIONS

| Risque | Impact | Mitigation |
|--------|--------|------------|
| Audiences trop petites pour TikTok (<50K) | Pas de delivery | Elargir interets, enlever contraintes age |
| Pixel pas assez alimente (<50 events/semaine) | Pas d'optimisation conversion | Rester en Traffic/LPV plus longtemps, basculer conversion a 50+ events |
| Creative fatigue rapide (petit budget = repetition) | CPM explose, CTR chute | Refresh creatif toutes les 2 semaines, recycler angles |
| iOS privacy (30-40% signal perdu) | Attribution incomplete | CAPI obligatoire, UTM + GA4 en backup, comparer donnees plateforme vs CRM |
| Saisonnalite ete = BBQ boom mais aussi vacances | Volume variable | Augmenter budget M4-M5 (mai-juin), reduire M7-M8 si besoin |
| Frais de port dissuasifs | Taux conversion LP faible | Tester offre "livraison offerte des 35 EUR" ou pack decouverte |

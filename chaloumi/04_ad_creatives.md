# 1. HOOKS VIDEO -- Les 3 premieres secondes

Les hooks sont classes par type psychologique. Tourner chaque hook en format vertical (9:16), smartphone, lumiere naturelle.

---

### HOOK 1 -- Curiosite sensorielle
- **Texte overlay** : "Ce fromage ne fond JAMAIS."
- **Action visuelle** : Gros plan du Chaloumi pose sur un gril brulant. La camera reste fixe 2 secondes. Le fromage grille, marques dorees, mais garde sa forme parfaite.
- **Pourquoi ca arrete le scroll** : Contre-intuitif. Tout le monde sait que le fromage fond. Le cerveau veut verifier.

### HOOK 2 -- Satisfaction visuelle (ASMR food)
- **Texte overlay** : "Le son de la perfection."
- **Action visuelle** : Son amplifie du Chaloumi qui touche la poele chaude -- gresillements nets. Fumee qui monte. Pas de musique, juste le son brut.
- **Pourquoi ca arrete le scroll** : Le gresillage food est le son le plus engageant sur TikTok/Reels. Declencheur sensoriel immediat.

### HOOK 3 -- Provocation / debat
- **Texte overlay** : "Le halloumi que vous achetez n'est PAS du vrai halloumi."
- **Action visuelle** : Main qui jette un paquet de halloumi industriel sur une table, puis sort un Chaloumi emballe artisanalement. Geste theatral.
- **Pourquoi ca arrete le scroll** : Attaque une croyance etablie. Cree un micro-conflit qui demande resolution.

### HOOK 4 -- Education rapide
- **Texte overlay** : "100% chevre = zero ballonnement."
- **Action visuelle** : Split screen. A gauche : liste d'ingredients d'un halloumi industriel (longue). A droite : liste Chaloumi (3 ingredients). Zoom progressif.
- **Pourquoi ca arrete le scroll** : Benefice sante direct, formulation simple, contraste visuel fort.

### HOOK 5 -- FOMO / rarete
- **Texte overlay** : "200 pieces par semaine. Pas une de plus."
- **Action visuelle** : Plan large de la chevrerie a Chypre, soleil levant. Puis coupe sur les mains du producteur qui moule le fromage. Compteur "Lot #047" en bas.
- **Pourquoi ca arrete le scroll** : Rarete reelle + artisanat visible = urgence authentique, pas marketing artificiel.

### HOOK 6 -- Transformation / avant-apres
- **Texte overlay** : "Avant / Apres 2 minutes sur le gril."
- **Action visuelle** : Chaloumi blanc, brut, pose a gauche. Time-lapse accelere sur le gril. Resultat dore a droite. Split screen synchronise.
- **Pourquoi ca arrete le scroll** : Le format avant/apres est un des plus performants en engagement. La transformation est visuellement satisfaisante.

### HOOK 7 -- Question directe
- **Texte overlay** : "Vous connaissez le fromage qu'on grille comme un steak ?"
- **Action visuelle** : Quelqu'un retourne le Chaloumi sur un BBQ avec une pince a viande, exactement comme un steak. Marques de grille visibles.
- **Pourquoi ca arrete le scroll** : Question = engagement cognitif automatique. L'analogie steak intrigue les BBQ lovers.

### HOOK 8 -- Temoignage brut
- **Texte overlay** : "J'ai goute et j'ai commande 3 fois depuis."
- **Action visuelle** : Visage face camera, bouche pleine, expression de plaisir authentique. Zero filtre, zero maquillage. Cuisine personnelle en arriere-plan.
- **Pourquoi ca arrete le scroll** : Authenticite UGC. Le visage humain avec une emotion forte capte l'attention plus que n'importe quel packshot.

### HOOK 9 -- Geographie / voyage
- **Texte overlay** : "Directement de cette ferme a Chypre."
- **Action visuelle** : Drone (ou smartphone en hauteur) survolant la chevrerie avec les montagnes de Chypre. Coupe rapide sur un colis qui arrive a une porte francaise.
- **Pourquoi ca arrete le scroll** : Depaysement instantane. Le contraste Chypre/France cree une histoire en 3 secondes.

### HOOK 10 -- Defi / interpellation
- **Texte overlay** : "Essayez de le faire fondre. On vous met au defi."
- **Action visuelle** : Chaloumi place dans une poele a feu maximum. Camera fixe. Le fromage grille, dore, mais reste intact. Texte "IMPOSSIBLE" qui apparait.
- **Pourquoi ca arrete le scroll** : Le defi active la curiosite competitive. Le spectateur veut voir le resultat.

# 4. BRIEFS UGC -- Pour createurs

---

## BRIEF 1 -- Food Blogger (Instagram/TikTok)

**Profil createur** : Food blogger francais(e), 10K-50K abonnes, contenu cuisine maison aesthetique. Pas de chef etoile -- cuisine accessible.

**Format** : Reel vertical 9:16, 20-30 secondes
**Concept** : "Taste test decouverte" -- le createur recoit le colis, decouvre le produit, le cuisine, le goute pour la premiere fois.

**Deroulement obligatoire** :
1. (0-3s) Ouverture du colis. Reaction authentique a l'emballage artisanal. Hook : "On m'a envoye un fromage de Chypre..."
2. (3-10s) Lecture des ingredients a voix haute (montrer qu'il n'y en a que 3). Mentionner "100% chevre".
3. (10-20s) Cuisson : le griller a la poele ou sur plancha. Insister sur le fait qu'il ne fond pas. Montrer la reaction "wow".
4. (20-25s) Degustation. Reaction faciale naturelle. Decrire le gout avec ses mots (pas de script mot a mot).
5. (25-30s) CTA naturel : "Le lien est dans leur bio / Je vous mets le lien en story."

**Points cles a mentionner** (au moins 2 sur 4) :
- Ne fond pas a la cuisson
- 100% lait de chevre
- Fait main a Chypre
- Livre en 48h

**Do's** :
- Filmer dans sa propre cuisine, lumiere naturelle
- Reaction genuinement surprise/enthousiaste (ne pas surjouer)
- Montrer le gresillage dans la poele (son naturel, pas de musique a ce moment)
- Utiliser ses propres mots, pas un script mot a mot

**Don'ts** :
- Pas de comparaison negative directe avec une marque nommee
- Pas de claims sante ("guerit", "soigne", etc.)
- Ne pas dire "pub" ou "sponsorise" dans le hook (le declarer selon la loi mais pas dans les 3 premieres secondes)
- Pas de filtre excessif sur la nourriture

---

## BRIEF 2 -- Couple BBQ (YouTube Shorts / TikTok)

**Profil createur** : Couple 28-40 ans, contenu lifestyle/BBQ/cuisine exterieure. Ambiance ete, jardin, convivialite.

**Format** : Reel vertical 9:16, 15-25 secondes
**Concept** : "BBQ du dimanche" -- integrer le Chaloumi dans une session BBQ entre amis ou en couple.

**Deroulement obligatoire** :
1. (0-3s) Plan du BBQ allume, ambiance ete. Hook : "On a trouve LE fromage pour le barbecue."
2. (3-8s) Poser le Chaloumi sur la grille au milieu des viandes/legumes. Contraste : c'est un fromage sur un BBQ, ca intrigue.
3. (8-15s) Le retourner. Montrer les marques de grille. Insister : "Il tient, il fond pas." L'autre personne reagit.
4. (15-20s) Degustation a deux. Brochette ou directement de la grille a l'assiette.
5. (20-25s) CTA : "Serieusement, commandez-le. Lien dans la bio de @chaloumi."

**Points cles a mentionner** (au moins 2 sur 4) :
- Tient parfaitement au BBQ/gril
- Fromage artisanal de Chypre
- Ingredient star de l'ete
- Direct du producteur

**Do's** :
- Filmer en exterieur, lumiere naturelle d'ete
- Montrer de la convivialite (amis, couple, famille)
- Son naturel du BBQ (gresillements, conversations)
- Format decontracte, pas de mise en scene lourde

**Don'ts** :
- Pas de sur-production (pas de drone, pas de montage complexe)
- Ne pas cacher les chevres/Chypre -- montrer l'origine si on a les visuels
- Pas de musique trop forte qui couvre les sons de cuisson
- Ne pas comparer avec une marque concurrente nommee

---

## BRIEF 3 -- Chef amateur / Recette (Instagram Reels)

**Profil createur** : Createur(rice) recettes, contenu top-down ou face camera, 5K-30K abonnes, style "recettes faciles".

**Format** : Reel vertical 9:16, 25-35 secondes
**Concept** : "Recette express au Chaloumi" -- une recette complete en moins de 30 secondes avec le Chaloumi comme ingredient star.

**Deroulement obligatoire** :
1. (0-3s) Hook visuel : ingredients etales sur plan de travail, Chaloumi au centre. "Cette recette va changer vos salades."
2. (3-10s) Prep : couper le Chaloumi en tranches, preparer les accompagnements (pasteque/menthe OU miel/roquette OU legumes grilles).
3. (10-18s) Cuisson du Chaloumi. Gros plan sur la poele/plancha. Montrer les deux faces dorees.
4. (18-25s) Dressage de l'assiette. Top-down. Geste du filet d'huile d'olive ou de miel.
5. (25-30s) Degustation + CTA : "La recette est dans la legende. Le fromage, c'est @chaloumi, lien en bio."

**Points cles a mentionner** (au moins 2 sur 4) :
- Le nom "Chaloumi" (pas juste "halloumi")
- Lait de chevre
- Ne fond pas
- Le tag @chaloumi + "lien en bio"

**Do's** :
- Filmer en top-down pour les etapes de preparation
- Transitions rapides (jump cuts sur les etapes)
- Texte overlay avec les ingredients et quantites
- Musique trending du moment (le createur choisit)

**Don'ts** :
- Pas de recette trop complexe (max 5 ingredients au total)
- Pas de plan de plus de 5 secondes (rythme rapide)
- Ne pas noyer le Chaloumi parmi trop d'ingredients -- il est la star
- Pas de voix off monotone -- energie positive ou ASMR, pas entre les deux

## 5.2 Planning de test -- 5 semaines

### SEMAINE 1 -- Validation du meilleur hook
**Objectif** : Identifier le hook qui genere le meilleur taux de retention 3 secondes.
- Lancer 4 videos avec le meme corps (Script A) mais 4 hooks differents : Hook 1, Hook 2, Hook 6, Hook 10
- Budget : reparti equitablement (25% chacun)
- Plateforme : TikTok Ads + Meta Reels
- Metrique cle : taux de retention 3 secondes (ThruPlay / Hook Rate)
- Seuil de decision : 500 impressions minimum par variation avant lecture

### SEMAINE 2 -- Validation du meilleur angle
**Objectif** : Trouver quel angle creatif performe le mieux en CTR et CPC.
- Garder le hook gagnant de S1
- Tester 3 angles : A1 (satisfaction gril), A3 (comparaison), A4 (recette)
- Meme audience, meme budget par angle
- Plateforme : Meta (Reels + Feed) + TikTok
- Metrique cle : CTR (link click), CPC, taux d'ajout au panier si pixel actif

### SEMAINE 3 -- Validation du meilleur copy
**Objectif** : Identifier le primary text qui genere le plus de conversions.
- Garder le hook gagnant S1 + angle gagnant S2
- Tester 3 primary texts : PT1 (emotionnel), PT2 (rationnel), PT3 (social proof)
- Version longue vs version courte du gagnant
- Plateforme : Meta (Facebook + Instagram)
- Metrique cle : taux de conversion (achat ou ajout panier), ROAS

### SEMAINE 4 -- Scale du gagnant + lancement UGC
**Objectif** : Scaler la combinaison gagnante et introduire le contenu UGC.
- Combinaison gagnante S1+S2+S3 : augmenter le budget x3
- En parallele, lancer les 2 premiers UGC (Brief 1 + Brief 2) en tant que nouvelles variations
- Plateforme : Meta (Advantage+ Shopping) + TikTok Spark Ads (si UGC sur compte createur)
- Metrique cle : ROAS, CPA, frequence (surveiller la fatigue)

### SEMAINE 5 -- Iteration + nouveaux angles
**Objectif** : Prevenir la fatigue creative et ouvrir de nouveaux segments.
- Si frequence > 3 sur le gagnant : introduire Script B (producteur) et Script E (logistique) comme relais
- Lancer les formats statiques : carrousel social proof (A6) + image urgence (A7)
- Tester une audience Lookalike basee sur les acheteurs S1-S4
- Metrique cle : CPA vs S4 (doit rester stable ou baisser), taux de nouvelles conversions

## 5.3 Criteres de decision (kill / scale / iterate)

| Resultat | Action |
|----------|--------|
| CTR > 2% et CPA dans la cible | SCALE : augmenter budget progressivement (+20%/jour max) |
| CTR > 2% mais CPA trop haut | ITERATE : garder le creatif, tester un nouveau CTA ou landing page |
| CTR < 1% apres 1000 impressions | KILL : couper la variation, analyser pourquoi |
| Frequence > 3.5 | ROTATE : introduire nouveau creatif, mettre l'ancien en pause |
| ROAS > 3x | SCALE FORT : dupliquer dans une nouvelle campagne dediee |
| UGC surperforme le brand content | PIVOTER : allouer 70% du budget UGC, 30% brand |

Format : `[Plateforme]_[Angle]_[Format]_[Hook#]_[Copy#]_[Version]`

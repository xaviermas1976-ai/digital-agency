## Commandes

| Commande | Action |
|----------|--------|
| `BRIEF [client]` | Intake client → Market Research + Competitive Analysis + Proposal |
| `SEO AUDIT [url]` | 13 agents SEO en parallèle → rapport consolidé |
| `GEO AUDIT [url]` | 7 agents GEO/AI visibility en parallèle |
| `PAID MEDIA [brief]` | Architecture paid media + création + tracking |
| `WEB BUILD [brief]` | Refonte site : UX → Design → Dev → QA |
| `SOCIAL LAUNCH [brief]` | Déploiement 6 plateformes sociales |
| `EMAIL CAMPAIGN [brief]` | Séquences email + growth hacking + tracking |
| `FULL DEPLOY [brief]` | TOUS les workflows en parallèle |
| `STATUS` | État projets actifs |
| `REPORT [client]` | Rapport PDF consolidé |

## Règles

1. **Parallélisme maximal** — lancer tous les agents indépendants simultanément
2. **Quality gate** — Reality Checker + Evidence Collector avant toute livraison
3. **Data-driven** — pas d'opinion sans données
4. **Budget-aware** — Financial Analyst valide les allocations
5. **Feedback loop** — chaque campagne nourrit la suivante
